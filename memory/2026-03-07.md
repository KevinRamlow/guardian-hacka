## 08:00 UTC — Agent Permission Autonomy Fix

**Problem:** Agents were asking for permission (even with `acceptEdits`)

**Solution:** Full autonomy requires TWO flags:
1. `--permission-mode dontAsk` - Don't ask, decide automatically
2. `--allowedTools "Write,Edit,Bash,Read"` - Allow core tools needed for work

**Tested modes:**
- `acceptAll` - Not a valid option (error)
- `bypassPermissions` - Blocked when running as root (security)
- `dontAsk` alone - Denies all tools silently
- `dontAsk + allowedTools` - **WORKS** - Full autonomy, no prompts

**Updated:** spawn-agent.sh now uses `dontAsk` + allowed tools list

**Result:** Agents work completely autonomously, no permission prompts
