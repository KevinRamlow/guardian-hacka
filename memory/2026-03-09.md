# 2026-03-09 — Unified Task Management Refactor

## What Changed

Complete overhaul of my task management. Replaced 50+ scripts and 2 registries with a unified state machine.

## New Architecture: state.json + 5 Core Scripts

**Single source of truth:** `/Users/fonsecabc/.openclaw/tasks/state.json`

**State machine:**
```
todo → agent_running → [done | failed | eval_running]
eval_running → callback_pending → agent_running → ...
```

### Core Scripts
| Script | Purpose |
|---|---|
| `task-manager.sh` | State CRUD + transitions (replaces agent-registry.sh + process-manager.sh) |
| `dispatcher.sh` | Create Linear task + register + spawn (replaces dispatch-task.sh) |
| `supervisor.sh` | Unified 30s launchd (replaces watchdog + process-checker) |
| `reporter.sh` | Report to Linear + Slack + dashboard (replaces agent-report.sh + agent-peek.sh) |
| `spawn-agent.sh` | Low-level spawner (still used, now calls task-manager.sh) |

### Launchd: 2 jobs instead of 9
| Job | Interval | Replaces |
|---|---|---|
| `com.anton.supervisor` | 30s | watchdog + process-checker |
| `com.anton.infra` | 15min | linear-sync + langfuse-scraper + gcp-token-push |

### Feedback Loop (NEW — first-class citizen)
Each task carries `history[]` and `learnings[]`. Callback agents get full context of previous cycles.
```bash
bash scripts/task-manager.sh add-history AUTO-XX '{"cycle":1,"accuracy":78.5,"delta":"+2.5pp"}'
bash scripts/task-manager.sh add-learning AUTO-XX "severity thresholds had more impact"
```

### Commands I Should Use
```bash
# Dispatch work
bash scripts/dispatcher.sh --title "Fix X" --desc "Details" --label Bug

# Check state
bash scripts/task-manager.sh list
bash scripts/task-manager.sh get AUTO-XX

# For evals
bash scripts/task-manager.sh transition AUTO-XX eval_running \
  --process-pid $PID --process-type eval --context "what I changed"
# Then EXIT

# Monitor
bash scripts/reporter.sh peek
bash scripts/reporter.sh peek AUTO-XX follow
```

### What I Must NOT Do
- Poll evals in loops (FORBIDDEN)
- Use sleep+check patterns (FORBIDDEN)
- Stay alive waiting for a process (FORBIDDEN)
