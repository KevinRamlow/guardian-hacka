## Tips for Better Results

1. **Be specific about format:** Always mention aspect ratio and intended platform
2. **Describe the mood:** Professional, casual, funny, elegant, energetic, etc.
3. **Mention style references:** "like tech startup style", "corporate professional", "meme format"
4. **Include composition details:** "centered", "split screen", "space for text at top"
5. **Specify colors when important:** "blue and white", "dark theme", "vibrant gradient"

## Iterating on Results

If the first result isn't perfect:
- Add more specific details to the prompt
- Mention what to change: "same but with darker colors"
- Use reference images when available
- Try different models: flash for speed, pro for quality

## Combining with Image Editing

Generate a base image, then use `edit_image` to refine:
